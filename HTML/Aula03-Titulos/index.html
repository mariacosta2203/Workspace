HTML
O HTML é considerado uma linguagem de marcação, pois o mesmo não pode fazer estrutura de dados, manipular variáveis, etc... O HTML nos acompanha para estruturar e formatar os conteúdos de página web, como: ordem dos textos, divisões das páginas, divisões de conteúdo, aplicar imagens, interligar uma página na outra etc... O HTML é considerado a estrutura de um site, muitas das vezes usam o exemplo de corpo humano, onde o HTML são os ossos, pois é os ossos que dão a estrutura do corpo humano.
CSS
HTML é uma linguagem de marcação e depois de criarmos a estrutura do nosso site, entra em cena o CSS, que é uma linguagem de estilização. A mesma nos traz a possibilidade de colocar cores, bordas coloridas, mexer no tamanho de imagens e figuras. Assim trazendo um visual melhor e atraente para o usuário.
JavaScript
JavaScript é uma linguagem de programação, pois é uma linguagem interativa com o usuário já que utilizamos a lógica na mesma, por exemplo: calculadora, colocar a seta do computador encima de alguma palavra e ela mudar de cor, etc.
